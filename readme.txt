== Description ==

Simple Admin Columns is very easy to use WordPress plugin. It helps you to create `custom columns` on the admin screens for posts and pages.

You can create a `Modified Date` sortable column and `Post Thumbnail` column for posts and pages in WordPress admin area by using a friendly interface.

= Features =
* Simple Admin Columns is very easy to use.
* You can create a sortable `Last Modified` or `Modified Date` column on posts and pages screen.
* You can add `Featured Post Thumbnail` column on posts and pages screen.

= About =
Simple Admin Columns plugin is developed by [WPFlask](https://wpflask.com/). You can use our free WordPress themes for your blog.

== Installation ==

= Using The WordPress Dashboard =

1. Navigate to the `Add New` in the plugins dashboard
2. Search for `Simple Admin Columns`
3. Click `Install Now`
4. Activate the plugin on the Plugin dashboard
5. Configure the plugin by going to the `Settings > Simple Admin Columns` options.

= Uploading in WordPress Dashboard =

1. Navigate to the `Add New` in the plugins dashboard
2. Navigate to the `Upload` area
3. Select `simple-admin-columns.zip` from your computer
4. Click `Install Now`
5. Activate the plugin in the Plugin dashboard
6. Configure the plugin by going to the `Settings > Simple Admin Columns` options.

= Using FTP =

1. Download `simple-admin-columns.zip`
2. Extract the `simple-admin-columns` directory to your computer
3. Upload the `simple-admin-columns` directory to the `/wp-content/plugins/` directory
4. Activate the plugin in the Plugin dashboard
5. Configure the plugin by going to the `Settings > Simple Admin Columns` options.
